# Корзина интернет-магазина

Консольное приложение для работы с корзиной покупок.

## Описание

Проект реализует систему корзины интернет-магазина с возможностью:
- Добавления товаров
- Удаления товаров по названию
- Просмотра содержимого корзины
- Подсчета общей суммы покупок
